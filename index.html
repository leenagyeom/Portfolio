안녕하세요!
개발과 계발을 게을리하지 않는 개발자 이나겸 입니다.

임베디드 연구원을 시작으로 저의 개발인생이 시작되었습니다.
임베디드 개발을 할 때는 C언어와 C++을 주로 사용했었고,
ATmega128과 ATmega328p를 비롯한 AVR 시리즈와 아두이노, 라즈베리파이를 사용했습니다.

배움에 대한 두려움이 없기에 개발 교육 경력도 가지고 있습니다.
직업훈련기관에서 강의를 할 때는 C언어와 Python을 주로 사용했습니다.
C언어를 주로 다뤘던 제가 Python 강의를 할 수 있었던 것은 끊임없이 공부를 한 덕분이었습니다.
Cortex-M4 코어를 사용하는 STM보드인 Nucleo-F429ZI 보드와 Raspberry PICO를 사용해보았고,
JAVA, HTML, CSS, JavaScript, Dart, C# 언어를 사용해보았습니다.

인공지능 과정을 맡아 운영하게 되면서 독학의 한계를 느끼는 계기가 되었습니다.
운영하던 반이 수료할 때 까지 인공지능 기술에 대한 갈증은 해소되지 않았고,
대구AI스쿨에서 인공지능 교육을 한다는 것을 알게된 후 인공지능 수업을 들을 수 있었습니다.

인공지능 교육을 들으면서 머신러닝과 딥러닝에 대해 익힐 수 있었으며
PyTorch Framework를 사용하여 다양한 CNN 모델로 Classification 프로젝트와
Yolov5, Yolov7을 사용하여 Object Detection 프로젝트를 진행하였습니다.

개발자가 되겠다고 다짐한 2018년부터 지금까지 계속 부족한 부분을 공부하며 채워가고 있습니다.
앞으로도 개발을 하면서 계속해서 새로운 것을 공부하고 적용할 수 있는
확실히 어제보다 나은 내일을 기대할 수 있는 개발자가 되겠습니다.
